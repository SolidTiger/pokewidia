Data

Pokéwidia is powered by PokéApi which provides data from the pokemon games.
A big thanks to the creators of the API. The API allows us to fetch JSON objects
containing data about specific Pokémon. The data we use are specific Pokémons stats, type, name, and ID.
You can find the API and more info at: https://pokeapi.co/
